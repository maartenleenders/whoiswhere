## User manual

Make sure you have docker installed.

In the root:
```bash
$ docker-compose up -d
$ cd client
$ yarn && yarn start
```

You can see the app running at `http://localhost:3001`

Enjoy!

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
